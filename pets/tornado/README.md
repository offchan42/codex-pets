# Tornado

A living kanban tornado that turns urgent work into done work.

This is a Codex pet v2 package with a `1536x2288` RGBA WebP atlas using `192x208` cells.

- Rows `0-8` preserve the approved legacy standard animations.
- Rows `9-10` add 16 clockwise look directions from `000 up` through `337.5 up-left`.
- `spriteVersionNumber: 2` enables the directional contract.
- The final atlas passed v2 structure, transparency, chroma, blind cardinal, labeled semantic, continuity, and visual QA.

See `qa/` for the final contact sheet, focused direction sheet, blind-direction result, continuity measurements, despill report, and validation result.
