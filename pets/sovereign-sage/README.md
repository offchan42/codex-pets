# Sovereign Sage

A serene rationality companion for lucid calm and reflective focus.

## Package

- `pet.json` - Codex pet manifest.
- `spritesheet.webp` - Final transparent-capable sprite atlas.
- `qa/contact-sheet.png` - Visual row review.
- `qa/review.json` - Extracted-frame inspection output.
- `qa/validation.json` - Atlas validation output.

## Notes

Sovereign Sage uses a meditative, mostly front-facing silhouette. `running-left` was derived by mirroring `running-right` because the design has no handed prop, readable text, or side-specific mark that would become semantically wrong.
